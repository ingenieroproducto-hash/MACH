# MACH para Android

Aplicación Android para el seguimiento operativo y financiero de entregas.

## Compilación

La acción `Compilar APK de MACH` genera automáticamente un APK de instalación.

## Almacenamiento

Los datos se guardan automáticamente dentro del almacenamiento privado de la aplicación. Antes de desinstalar o borrar datos, exporta un respaldo JSON desde MACH.
